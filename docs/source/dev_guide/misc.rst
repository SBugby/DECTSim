Miscellaneous
=============

rotz
----

.. function:: rotz(angle)

   :param angle: The angle of rotation in radians.
   :type angle: double

   Return a 3x3 rotation matrix for a rotation about the z-axis.

roty
----

.. function:: roty(angle)

   :param angle: The angle of rotation in radians.
   :type angle: double

   Return a 3x3 rotation matrix for a rotation about the y-axis.
